##### 텍스트를 강조하는 방법
- - - - -
+ **굵게** 표현하는 방법은 이렇게 하거나
+ 또는 __이렇게 하면 BOLD로 표현된다__
+ 글자를 *기울이는* 방법은
+ 이렇게 중간에 _언더바_ 로 감싸면 된다.
+ 
  + 긁으면서도 기울이게 하려면 ***이렇게 하거나***
  + ___이렇게___ 하면 된다.
 
취소선은 ~~이렇게~~ 하면 표현이 가능합니다.
